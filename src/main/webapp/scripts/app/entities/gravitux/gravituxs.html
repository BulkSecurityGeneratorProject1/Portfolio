GraviTux Javascript

(60/60 fps)

Info

GraviTux JS is a reimplementation of GraviTux in JavaScript. GraviTux is a Java game developed by fellow students and me back in 2012.
The Sourcecode of GraviTux and GraviTux JS is available on GitHub.

How to play

You play as the linux mascot Tux in a penguin world and have to collect fish barrels. In order to succeed, you need to control the gravity.
You can run with W-A-S-D or the arrow keys. Use SPACE to swap gravity.
Have Fun!
